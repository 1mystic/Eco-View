How to Use Each Session Window

Add exactly 2 files as context: MASTER_PLAN.md + the current session file (e.g., session_01_data.md)
Say: "Execute this session. Write the code."
When done, fill in the corresponding outputs/sN_*.txt file
Move to the next session
One Decision I Made For You
Tabular metadata = 4 seasons + 5 AQI bins (9 features, fully synthetic). This requires zero external APIs — the vector is generated from the image labels using simple probability rules (Air_Pollution images → higher AQI bins, Clean images → lower). This is documented in session_01_data.md.

Start with Session 01 whenever you're ready.
